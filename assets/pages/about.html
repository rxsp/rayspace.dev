About This Site

The aesthetic design of this website draws inspiration from the functional and clean layout designed by Lee Robinson but diverges in technical stack.

I constructed the frontend from scratch with just the foundational web technologies - HTML, CSS, and JavaScript. This choice was deliberate, I wanted to avoid the use of any frontend frameworks to maintain a minimalist approach not only in design but also in the choice of technologies.

The backend tells a similar story. I built a hyper-efficient web server using Rust, a language renowned for its speed and memory safety. With the combination of Actix Web and PostgreSQL, I'm able to efficiently handle REST endpoints and other backend operations.

This site is fully open source and featured in the official Actix Web Examples. To learn more about my thoughts on applying a minimalist approach to web development, you can read my blog post titled "Minimalism in Web Development".
